# Double Negative

Double Negative 브랜드 랜딩 페이지 — Vite + React + Tailwind CSS + Framer Motion

## 실행 방법

```bash
cd double-negative
npm install
npm run dev
```

브라우저에서 `http://localhost:5173` 으로 접속하세요.

## 스크립트

- **`npm run dev`** — 개발 서버 실행 (HMR)
- **`npm run build`** — 프로덕션 빌드 (`dist/`)
- **`npm run preview`** — 빌드 결과물 미리보기

## 기술 스택

- **Vite** — 빌드 도구
- **React 18** — UI
- **Tailwind CSS** — 스타일링
- **Framer Motion** — 애니메이션
- **Lucide React** — 아이콘
